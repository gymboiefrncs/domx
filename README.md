# DomX

Code snippet sharing platform with group-based collaboration, real-time posting, markdown rendering, and syntax highlighting.

## For Users

### What You Can Do

- Sign up with email + OTP verification.
- Complete profile setup (username + password).
- Create groups and add members by display ID.
- Share code snippets in real time inside group chats.
- Edit and delete your own snippets.
- View markdown-formatted snippets with code highlighting.

### Main User Flow

- Create account: `Signup -> OTP -> Set Info -> Login`.
- Join or create a group.
- Open a group chat and post snippets.
- Manage membership and group settings (if authorized).
